Projects

Python Chess Game

For my Object-Oriented Programming course, I developed a complete two-player chess game using Python and Pygame. This project was a great opportunity to aquire OOP skills, as I used concepts like inheritance and polymorphism to represent the different chess pieces and their unique movements. I also implemented design patterns like the Factory Method to create the pieces. Beyond just moving pieces, the game enforces all the rules of chess, including tricky ones like en passant and castling. I also added features like customizable timers and a log file that saves game stats after each match. This project wasn't just about fulfilling a course requirement, it was a chance to build a complete, playable game and solidify my understanding of Python and software design principles.
